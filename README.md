Coal Mine Safety Monitoring and Alerting System
📌 Project Overview

The Coal Mine Safety Monitoring and Alerting System is designed to improve safety conditions inside coal mines. The system continuously monitors environmental parameters such as gas levels, temperature, and humidity. If any parameter exceeds safe limits, the system immediately alerts workers and supervisors to prevent accidents.

🎯 Aim

To develop a smart monitoring system that detects hazardous conditions in coal mines and provides real-time alerts to improve miner safety.

📋 Objectives

Monitor gas concentration levels inside coal mines.

Measure temperature and humidity in the mining environment.

Detect dangerous conditions and send alerts.

Provide real-time monitoring to prevent accidents.

🧱 Block Diagram

The system consists of sensors connected to a microcontroller that collects environmental data. When unsafe conditions are detected, an alert is generated through buzzer or notification system.

🔍 Working Principle

Sensors collect environmental data from the mine.

The microcontroller processes the sensor readings.

If any value exceeds the safety threshold:

A buzzer alarm is activated.

An alert is sent to the monitoring system.

The system helps miners take immediate action.

📊 Advantages

Improves worker safety.

Provides real-time monitoring.

Early detection of hazardous conditions.

Reduces risk of mine accidents.

⚠️ Limitations

Requires reliable power supply.

Sensor calibration is necessary.

Wireless communication may be limited underground.
